# Handoff Template

Copy this structure when work is incomplete or when another agent needs to pick
up quickly.

## Current State

- What was completed
- What is still in progress
- What should not be rebuilt

## Files Touched

- List the important files changed

## Checks Run

```bash
pnpm lint
pnpm typecheck
pnpm build
```

Note any failures or skipped checks.

## Next Step

- State the single highest-value next move

## Notes For The Next Agent

- Include design decisions, edge cases, placeholders, or open questions
